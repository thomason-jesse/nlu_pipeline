Add a word to the pronunciation dictionary. 

This function adds a word to the pronunciation dictionary and the current language model (but, obviously, not to the current F\-S\-G if F\-S\-G mode is enabled). If the word is already present in one or the other, it does whatever is necessary to ensure that the word can be recognized.


\begin{DoxyParams}{Parameters}

\hline
{\em phones} & Whitespace-\/separated list of phoneme strings describing pronunciation of {\ttfamily word}. \\
\hline
{\em update} & If T\-R\-U\-E, update the search module (whichever one is currently active) to recognize the newly added word. If adding multiple words, it is more efficient to pass F\-A\-L\-S\-E here in all but the last word. \\
\hline
\end{DoxyParams}
