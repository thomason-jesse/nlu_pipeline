\subsection{Detailed Description}
Apply Cepstral Mean Normalization (C\-M\-N) to the set of input mfc frames. By subtractingthe mean of the input from each frame. C0 is also included in this process. This function operates on an entire utterance at a time. Hence, the entire utterance must be available beforehand (batchmode).
